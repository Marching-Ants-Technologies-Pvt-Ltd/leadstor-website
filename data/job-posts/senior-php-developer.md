## Senior PHP Developer

**Experience:** 4 to 8 years &bullet; **Opportunity:** Remote/WFH &bullet; **Placement:** Full time Permanent Position &bullet; **Skills:** PHP, MySQL, JavaScript, HTML, CSS, Laravel, CodeIgniter, RESTful APIs, MVC architecture, problem-solving, debugging, code optimization, Git, teamwork, communication skills, leadership, project management, unit testing, database design, performance tuning.

We are seeking an experienced Senior PHP Developer to join our dynamic team. The ideal candidate will have 4 to 8 years of experience in PHP development, with a strong background in building high-quality web applications. As a Senior Developer, you will be responsible for leading development projects, optimizing application performance, and implementing new features while maintaining code quality and best practices. You will work closely with cross-functional teams to deliver scalable and robust solutions that align with business objectives.

### Key Responsibilities

- Lead the development, testing, and deployment of web applications using PHP and related technologies.
- Architect and design application structures, ensuring they are scalable, secure, and efficient.
- Collaborate with product managers, designers, and other developers to deliver high-quality products.
- Conduct code reviews, provide feedback, and mentor junior developers to foster a collaborative team environment.
- Optimize application performance, troubleshoot, and debug issues.
- Write clean, well-documented, and efficient code following industry standards.
- Stay updated with the latest industry trends and technologies to continuously improve development processes.

### Requirements

- 4 to 8 years of experience in PHP development.
- Proficiency in PHP frameworks such as Laravel or CodeIgniter.
- Strong understanding of MySQL and database design principles.
- Experience with front-end technologies like HTML, CSS, and JavaScript.
- Familiarity with RESTful APIs, MVC architecture, and Git version control.
- Strong problem-solving, debugging, and code optimization skills.
- Excellent communication and teamwork abilities.
- Experience with project management, unit testing, and performance tuning.
- Ability to lead and mentor junior developers effectively.
- Knowledge of modern development practices and a proactive approach to learning new technologies.
